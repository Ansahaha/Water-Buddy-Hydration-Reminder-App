
# 🐱 Water Buddy

Water Buddy is a mobile hydration reminder application designed to help users maintain healthy water intake habits throughout the day.

The application features a friendly cat companion named Poco who motivates users to stay hydrated by tracking their daily progress in a fun and engaging way.

---

## 📱 Project Overview

Many people forget to drink enough water during busy schedules. Water Buddy solves this problem by providing a simple and interactive hydration tracking experience.

Users can log each glass of water they drink and visually monitor their progress through a water-filled progress circle.

As hydration levels increase, Poco reacts with different emotions and animations, creating a rewarding user experience.

---

## ✨ Features

- Daily water intake tracking
- Visual hydration progress indicator
- Percentage-based completion system
- Friendly mascot character (Poco)
- Interactive user experience
- Simple and clean interface
- Motivation through character reactions
